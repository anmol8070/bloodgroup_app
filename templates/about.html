About Our Project

The Blood Group Detection using Fingerprint system leverages image processing and machine learning (CNN) techniques to predict an individual's blood group based on fingerprint images.

Our goal is to provide a fast, non-invasive method for blood group detection that can assist hospitals and emergency services.

  • 🧠 Powered by: Convolutional Neural Network (CNN)
  • ⚙️ Backend: Python, Flask
  • 🎨 Frontend: HTML, CSS, JavaScript
  • 📸 Input: Fingerprint image (BMP format)
  • 🔍 Output: Predicted blood group (A, B, AB, O + / -)

This system can be integrated into hospital management software to streamline patient onboarding and emergency care.
